# gulp-template
This is simple boilerplate for gulp projects. It has browser reload and SCSS preprocessor and basic HTML and CSS markup with media queries and SCSS variables. Mobile first design.


Folder architecture: 

.
├── app
│   ├── assets
│   │   ├── css
│   │   │   └── main.css
│   │   ├── js
│   │   │   └── main.js
│   │   └── sass
│   │       └── main.scss
│   ├── images
│   └── index.html
├── dist
│   └── stylesheets
│       └── css
│           └── main.css
├── gulpfile.js
├── LICENSE
├── package.json
├── package-lock.json
└── README.md

How to use:

1) run npm install
2) run gulp watch
3) start coding
